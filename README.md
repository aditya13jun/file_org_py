# 📂 File Organizer using Python

A simple Python automation project that organizes files into folders based on their file extensions.

## 🚀 Features

- Organizes `.log` files
- Organizes `.pdf` files
- Organizes `.png` image files
- Organizes `.docx` files
- Automatically creates folders if they do not exist

---

## 🛠️ Technologies Used

- Python
- os module
- shutil module

---

## 📁 Project Structure

```text
D:/temp/
│
├── log files/
├── pdf files/
├── image files/
├── doc files/
```

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/aditya13jun/file_org_py.git
```

2. Open the project folder

3. Run the Python file

```bash
python file_organizer.py
```
